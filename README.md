### Descrición:
Este proyecto es una aplicación de calculadora interactiva ejecutada en consola. El programa solicita al usuario que ingrese dos números(enteros o decimales) y devuelve el resultado de las siguientes operaciones:
- Adiccion
- Sustraccion
- Multiplicacion
- Division
- Modulo
El programa tiene validacion para el intento de division por cero, y todo intento dara por resultado 0

Para la elaboracion de este programa se utilizo el lenguaje de programación Java y librerias del mismo como Java.Util
